# ETHEREAL Luxury Perfume Landing Page

A high-end, cinematic single-page landing page for a luxury perfume brand, built with **React**, **Vite**, and **Framer Motion**.

## ✨ Features
- **Cinematic Reveal**: Split-screen sticky panels inspired by modern luxury advertisements.
- **Scroll-Driven Parallax**: Smooth background zoom and text fades as you navigate.
- **Ultra-Modern UI**: Minimalist typography, gold accents, and deep black aesthetics.
- **Responsive Grid**: Interactive product collection with fluid hover effects.
- **Professional Structure**: Includes Testimonials, Newsletter signup, and a Detailed Footer.

## 🚀 Deployment

This project is configured for easy deployment to **GitHub Pages**.

1. Create a repository on GitHub named `perfume`.
2. Link your local project:
   ```bash
   git remote add origin https://github.com/deeptha00/perfume.git
   ```
3. Deploy to live:
   ```bash
   npm run deploy
   ```

## 🛠️ Development

Install dependencies:
```bash
npm install
```

Start the dev server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
